## Sobre o projeto

> Criação de uma aplicação e utilização o serviço de web static apps do Microsoft Azure para hospedagem e configuração de um site.

Neste projeto, focamos na criação e hospedagem de um site estático utilizando uma combinação de tecnologias modernas e acessíveis. Aproveitamos o Static Web App da Microsoft Azure para a hospedagem gratuita e confiável, enquanto o desenvolvimento foi realizado no Visual Studio. Para suporte adicional, utilizamos GitHub para controle de versão, Gatsby como framework de desenvolvimento e Node.js para gerenciamento de dependências.
